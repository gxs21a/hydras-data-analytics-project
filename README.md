# Hydras Group Project

Data Mining 2: Applied Machine Learning Methods

## Setup Instructions

1. Install [Python 3.14](https://www.python.org/downloads/windows/)

2. Install [Git](https://git-scm.com/install/windows)

3. Clone GitHub Repository

```
git clone https://github.com/gxs21a/my-github-stuff.git
```

4. Download Requirements

```
pip install -r requirements.txt
```
